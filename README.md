# customer_behaviour_analysis
This is a customer behavior analysis project built using SQL and PostgreSQL. The project focuses on analyzing customer data to understand revenue trends, product performance, and purchase patterns. All the analysis is performed using SQL queries executed in pgAdmin, helping to practice real-world data analysis and improve SQL querying skills.
